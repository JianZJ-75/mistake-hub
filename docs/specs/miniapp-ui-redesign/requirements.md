# 小程序 UI 全面重设计 — Anthropic 品牌色系 - Requirements Document

将微信小程序的视觉体系从冷色调蓝色（#4A7CF8）全面切换至 Anthropic 品牌暖色调（主色橙 #d97757、辅色蓝灰 #6a9bcc、成功绿 #788c5d），同时建立共享 SCSS 变量体系消除硬编码，优化圆角/间距/阴影等视觉细节，提升整体高级感。涉及 12 个现有 SCSS/TSX 文件改造 + 3 个新建工程文件。
