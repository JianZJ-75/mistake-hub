# 错题复习调度小程序

## 一、背景

随着信息技术在教育领域的不断发展，基于软件系统的学习辅助工具在学生自主学习过程中发挥着越来越重要的作用。错题复习作为巩固知识、提升学习效果的重要手段，在实际学习过程中却普遍存在复习随意性强、复习计划缺乏科学依据、错题重复遗忘等问题，导致学习效率难以有效提升。

艾宾浩斯记忆曲线揭示了人类记忆随时间衰减的客观规律，为科学制定复习计划提供了理论基础。当前部分学习类应用已尝试引入记忆曲线思想，但多数系统仍停留在固定时间提醒或简单复习提示层面，缺乏对错题复习过程的系统建模，未能在有限学习时间约束下实现复习任务的合理调度，难以满足个性化学习需求。

因此，设计并实现一个基于艾宾浩斯记忆曲线的错题复习调度小程序，对错题复习过程进行建模与调度优化，在保证系统可实现性的前提下提升复习效率与学习效果，具有一定的实际应用价值和工程实践意义。

## 二、任务

### 2.1 系统总体设计

1. 对错题复习业务进行需求分析，明确系统的主要使用对象为学生用户和系统管理用户，梳理错题采集、复习调度、复习执行和学习统计的整体业务流程；
2. 设计系统总体架构，采用前后端分离架构，构建由手机端微信小程序、电脑端 Web 管理系统以及后端服务组成的多端协同系统；
3. 后端采用 Spring Boot 框架实现核心业务逻辑，通过 RESTful API 为前端提供错题管理、复习调度、复习记录和数据统计等服务接口；
4. 采用 MySQL 作为系统主数据库，设计用户表、错题表、复习计划表、复习记录表等核心数据表，并建立合理的数据关联关系；
5. 引入 Redis 对高频访问数据（如当日复习任务列表、复习调度结果）进行缓存，提高系统整体响应性能。

### 2.2 手机端（微信小程序）用户登录与基础设置模块

1. 实现基于微信授权的用户登录功能，使用微信 openid 作为用户唯一标识，完成用户身份识别；
2. 实现用户个人信息展示功能，支持用户查看个人错题数量、复习次数及当前学习状态；
3. 实现基础学习参数设置功能，支持用户设置每日最大复习题量等个性化参数。

### 2.3 手机端错题采集与管理模块

1. 实现错题录入功能，支持学生通过表单方式录入错题题干、正确答案及错误原因说明等信息；
2. 实现错题图片上传功能，支持学生拍照或从相册选择题目图片作为错题附件保存；
3. 实现错题分类与标签管理功能，支持为错题选择所属学科、章节或知识点标签；
4. 实现错题列表展示与查询功能，支持按学科、知识点、掌握状态等条件对错题进行筛选和浏览；
5. 维护错题状态信息，记录错题的复习阶段、最近复习时间和掌握程度，为复习调度算法提供数据支持。

### 2.4 基于艾宾浩斯记忆曲线的复习调度算法模块

1. 基于艾宾浩斯记忆曲线理论，对错题复习过程进行抽象建模，为每道错题定义复习阶段、复习间隔和掌握度等状态参数；
2. 设计错题复习状态更新规则，根据学生每次复习的结果（答对或答错），动态调整错题的复习阶段和下一次复习时间；
3. 在每日复习题量受限的条件下，设计复习任务生成策略，从待复习错题中按照优先级筛选生成"今日复习任务列表"；
4. 将复习调度逻辑作为后端独立业务模块实现，保证复习计划生成过程的统一性和可维护性。

### 2.5 手机端复习执行与结果反馈模块

1. 实现"今日复习任务"页面，向学生展示系统自动生成的每日复习错题列表；
2. 实现错题复习交互功能，支持学生重新作答错题并提交复习结果；
3. 根据学生复习结果，实时更新错题的掌握度、复习阶段及下次复习时间；
4. 实现复习记录管理功能，完整记录每次复习的时间、结果及复习次数，形成复习行为数据闭环。

### 2.6 学习数据统计与分析模块

1. 实现学生个人错题统计功能，对错题总数、已掌握错题数和待复习错题数等指标进行统计；
2. 实现复习完成情况统计功能，对每日复习完成率和连续复习天数等数据进行分析；
3. 通过图表形式对学习数据进行可视化展示，帮助学生直观了解自身学习情况。

### 2.7 电脑端（Web）系统管理与分析模块

1. 实现系统管理端登录功能，用于系统维护和数据管理；
2. 实现用户数据管理功能，支持查看系统用户的基本信息及学习统计数据；
3. 实现错题数据管理功能，支持对错题数据进行查询和维护；
4. 实现复习调度参数配置功能，支持对复习间隔参数和每日最大复习题量等系统策略进行配置；
5. 实现系统运行日志查看功能，用于系统运行状态的基本监控与分析。

### 2.8 系统测试与运行分析模块

1. 对系统各功能模块进行功能测试，验证错题管理、复习调度和复习记录等功能的正确性；
2. 对复习调度算法进行效果分析，通过复习前后错题掌握情况变化验证算法的合理性；
3. 对系统运行性能进行测试，分析 Redis 缓存机制对系统响应性能的提升效果。
