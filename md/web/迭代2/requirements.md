# Web 管理端迭代2实施规划 - 需求文档

基于迭代2后端已完成的错题 CRUD 和标签管理接口，实现 Web 管理端的错题管理页面。

## Core Features

### 2.W1 错题管理页

- 表格展示当前用户的错题列表，支持分页
- 筛选条件：学科（下拉选择）、关键字（搜索框）、掌握度范围
- 每行显示：题干摘要、学科、掌握度进度条、标签、创建时间、操作按钮
- 操作：查看详情、删除（二次确认）

### 2.W2 错题详情/编辑弹窗

- 点击"详情"弹出 Dialog，展示完整错题信息
- 查看模式：只读展示题干、答案、错误原因、图片预览、标签列表、复习状态
- 编辑模式：点击"编辑"切换为表单，可修改题干、答案、错误原因、学科、标签
- 删除确认：点击"删除"弹出二次确认 Dialog

## User Stories

- 作为管理员，我可以在错题管理页看到自己的错题列表，并通过学科、关键字进行筛选
- 作为管理员，我可以查看某道错题的完整详情
- 作为管理员，我可以编辑错题的题干、答案、错误原因和标签
- 作为管理员，我可以删除错题（二次确认后软删除）

## Acceptance Criteria

- [ ] 错题管理页正确展示错题列表，分页切换正常
- [ ] 学科下拉筛选返回正确结果
- [ ] 关键字搜索支持题干模糊匹配
- [ ] 详情 Dialog 正确展示所有字段，图片可预览
- [ ] 编辑模式修改后保存成功，列表自动刷新
- [ ] 删除确认后错题从列表消失
- [ ] 侧边栏新增"错题管理"菜单项，可正常跳转

## Non-functional Requirements

- **UI 组件**：使用 shadcn/ui 组件库（Table、Dialog、Input、Select、Button 等），禁止引入 Ant Design
- **数据展示**：掌握度使用色标（红/黄/绿）直观展示
- **用户体验**：操作按钮在表格行内，避免多次跳转

## 依赖的后端接口

| 接口 | URL | 方法 | 说明 |
|------|-----|------|------|
| 错题列表 | `/v1/mistake/list` | POST | 分页查询，支持学科/标签/关键字/掌握度筛选 |
| 错题详情 | `/v1/mistake/detail` | POST | 返回完整错题信息（含标签列表） |
| 编辑错题 | `/v1/mistake/update` | POST | 更新错题信息和标签关联 |
| 删除错题 | `/v1/mistake/delete` | POST | 软删除错题 |
| 标签树 | `/v1/tag/tree` | POST | 编辑时标签选择使用 |
